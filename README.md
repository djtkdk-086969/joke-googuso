仕様サスペンス劇場
==================

Google の「未指定」を検出すると音を鳴らすジョークアドオン

## 概要
*これはエイプリルフールネタです。*

Googleの検索語句無視（未指定：）を検出すると音を鳴らします。

鳴らしたい音声ファイルは、アドオンとして発行する前に
`resources` フォルダの中に `killed.mp3` という名前で置いてください。

## これなら多分動くシステム要件
* Mozilla Firefox
* Google Chrome
